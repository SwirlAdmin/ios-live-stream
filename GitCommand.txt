Git Commands

1) git clone "https://github.com/SwirlAdmin/ios-sdk.git" (example)
2) Add files or project in ios-sdk folder
3) cd ios-sdk
4) git add . (For add files or project)
5) git commit -m 'Add Library' (For commit files or project)
6) git push origin main
7) git branch -a (For display branch name)
8) git branch -M master (For change branch)
9) git checkout master
